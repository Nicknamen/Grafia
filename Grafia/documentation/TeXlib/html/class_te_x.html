Detailed Description

This class is a TeX quick-compiler: it basically converts small portions of tex code to pdf or png. It is not designed to handle a proper tex file (reason for which it the methods to write and compile are implemented in the same class), even though, with little modification, it can be used as such.

◆ open() [1/2]

bool TeX::open ( )

Opens the file stored in _texname. It returns

  • false If no file constructor or opene(filenamed) has been called ever before or if any error occurs when opening the file
  • true If it actually opens a file and everything goes well.

No exception is thrown in this function, for it returns false easily. Exceptions must be handled with an if statement when calling the function.
